# PostgreSQL

## Mount / Volume

| Volume                 | Description       |
|------------------------|-------------------|
| `/etc/postgresql17/`   | Config Directory. |
| `/var/lib/postgresql/` | Data Directory.   |

## Network Ports

| Port        | Description     |
